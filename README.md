# Site

Landing page romântica construída com Next.js, TypeScript e Tailwind.

## Como rodar

1. Instale as dependências com `npm install`.
2. Inicie o projeto com `npm run dev`.

## Publicação

O projeto está configurado para export estático e deploy automático no GitHub Pages.

1. Crie um repositório no GitHub com qualquer nome.
2. Faça push para `main` ou `master`.
3. O site será publicado em `https://<seu-usuario>.github.io/<nome-do-repo>/`.
